## 项目目的

为了加强对计算机学院以及学校的宣传工作，在计忆传媒公众号的基础上，借助开源课的平台以及拥有的老师，学长指导等技术优势，基于discuz初步搭建计算机学院官方论坛，为计算机等专业学生提供了学科交互平台，为学院和学校的宣传工作注入新的活力。

## 项目特色

1、基础构建：discuz全球成熟度最高、覆盖率最大。

2、多资源宠重用：我和张京宇所在小组正在做一个市级大创项目，利用了学校的linux系统云服务器，局限性是只能使用校园网进行登陆，而Linux系统的优势为可跨平台、有丰富的软件支持、可多用户同时操作、安全性、稳定性好、网络功能完善。

3、移动支持：已开发Android的app手机客户端（qq群）fusion app

4、结合现实需求：本开源项目设立的目标并没有局限于开源课，而是结合新闻中心在学院中的特色定位，通过本次开源项目，为对计算机学院和北京信息科技大学的宣传注入新的活力。

## 遇到的主要问题与解决方案

1、xml环境不支持——修改阿帕奇服务器配置文件

2、服务器中文件的访问权限不够——终端指令 chmod 777 *（文件路径）

3、服务器不支持GD图片库的预览——修改阿帕奇服务器配置文件（用户无法上传头像）

4、修改界面时修改版权信息和连接部分，导致上传后数据库出现问题，每次做一定迭代前都会预留旧版本，重新安装预留版本。
